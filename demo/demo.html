Demo - Simple Grids

Features in this demo

  • Auto-creation of columns. Note this demo uses an alternate data-set with just 4 columns.
  • Switching the theme
This is as simple as it is. Two lines of griddl code to create a fully functional grid. 
// create a griddl.
var grid = new griddl.Grid('.griddl-container');

// get some data
var data = repository.getUsers();

// render your griddl, passing in the data
grid.render(data);

Demo - Custom formatting

Features in this demo

  • First name field formatted to show 'salutation first last'. Field still sorts by first (lines 5-7)
  • The whole model is available for custom formatting not just the column you are acting on
  • Data type of int, float and boolean (lines 4, 10, 13)
  • Format salary/float as currency (lines 10-12)
  • Automatically format boolean to yes/no
  • Change default items per page to 20 (line 1)
  • ID column does not allow sorting (line 4)
Custom column formatting is performed by passing a formatter function to griddl.Column 
var griddlSettings = {
    perPage: 20,
    columns: [
        new griddl.Column('id', 'ID', 10, griddl.sortType.NO_SORTING),
        new griddl.Column('firstname', 'Name', 30, griddl.sortType.STRING, function (employee) {
            return (employee.gender === 'male' ? 'Mr. ' : 'Ms. ') + employee.firstname + ' ' + employee.lastname;
        }),
        new griddl.Column('phone', 'Phone', 25, griddl.sortType.STRING),
        new griddl.Column('salary', 'Salary', 20, griddl.sortType.FLOAT, function (employee) {
            return '$' + employee.salary.toFixed(2).replace(/(\d)(?=(\d{3})+\.)/g, '$1,');
        }),
        new griddl.Column('isActive', 'Is Active', 15, griddl.sortType.BOOL)
        ]
    };
});

Demo - Custom sorting

Features in this demo

  • Last name field has a custom sorter. When you sort by last name the first name field will also sort (first within last). The custom sort is coded so the first name will always be sorted in ascending order even though last may be in decending order (line 9).
  • To use a custom sorter, pass a function instead of a griddl.sortType as the sorter parameter to the griddl.Column constructor
  • customSorters.lastName in the code sample below is the custom sort function. View source to see its details.
  • Starting with data already sorted. State field loaded in ascending order (line 2).
  • Selectable rows (line 5).
  • OnRowSelect callback. Click a row to see it work (line 6 and 14).
var colState = new griddl.Column('state', 'State', 15, griddl.sortType.STRING);
colState.currentSortOrder = griddl.columnSortOrder.ASC;

var griddlSettings = {
    selectableRows: true,
    keyField: 'id',
    columns: [
        new griddl.Column('firstname', 'First Name', 20, griddl.sortType.STRING),
        new griddl.Column('lastname', 'Last Name', 20, customSorters.lastName),
        new griddl.Column('street', 'Street', 25, griddl.sortType.STRING),
        new griddl.Column('city', 'City', 20, griddl.sortType.STRING),
        colState
    ],
    onRowSelect: function (rowKey) {
        alert('You clicked the row with id ' + rowKey);
    }
};

Demo - Custom Pager

Features in this demo

  • Use external pager. In this case, simplePagination.js
  • Place custom pager inside griddl's .gdl-panel-pager
  • Use griddl callbacks (onSort, onPerPageChange) to manage the custom pager
  • Use griddl onInit to set up custom pager
  • Call griddl's page method when custom pager gets clicked
var griddlSettings = {
    columns: [
        new griddl.Column('name', 'Name', 30, griddl.sortType.STRING),
        new griddl.Column('age', 'Age', 20, griddl.sortType.INT),
        new griddl.Column('gender', 'Gender', 20, griddl.sortType.STRING),
        new griddl.Column('company', 'Company', 30, griddl.sortType.STRING)
    ],
    onSort: function (column) {
        $('.griddl-container .gdl-panel-pager').pagination('drawPage', 1);
    },
    onPerPageChange: function (numberPerPage) {
        $('.griddl-container .gdl-panel-pager').pagination('updateItemsOnPage', numberPerPage);
        $('.griddl-container .gdl-panel-pager').pagination('drawPage', 1);
    },
    onInit: function (rowCount) {
        $('.griddl-container .gdl-panel-pager').pagination({
            items: rowCount,
            itemsOnPage: 10,
            onPageClick: function(pageNumber, event){
                grid.page(pageNumber);
            }
        });
    }
};

Demo - Custom CSS

Our philosophy:
We believe in separation of layout from code. Because of the desire to keep the code-base simple, we aren't giving you layout properties for every little display feature. You want the pager to the right instead of the left. Switch the float by overriding the css. Don't like the bare, spartan footer, want a little formatting. Go ahead, add some css.
See the readme for more details.

Demo - Grid Searching

Features in this demo

  • Grid searching. In this demo search by first or last name
  • Clearing search results
var griddlSettings = {
    searcher: function (employee, searchString) {
        var searchUpper = searchString.toUpperCase();
        return employee.lastname.toUpperCase().indexOf(searchUpper) !== -1 ||
            employee.firstname.toUpperCase().indexOf(searchUpper) !== -1;
    },
    columns: [
        new griddl.Column('firstname', 'First Name', 20),
        new griddl.Column('lastname', 'Last Name', 20),
        new griddl.Column('street', 'Street', 25),
        new griddl.Column('city', 'City', 20),
        new griddl.Column('state', 'State', 15)
    ]
};

$('.button-search').on('click', function () {
    grid.search($('search-criteria').val());
});
